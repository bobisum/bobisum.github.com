Slida

is a lite, fully responsive JQuery carousel plugin that uses CSS3 transitions if available.

It offers

a few (for now) options that the user can easily choose from.

Options:

There are 2 main options to choose from: autoanimate and elements count on stage (cool a?!).

No. of Elements

You can choose the numbers of the elements that you see on stage (default is 1), and by hitting the left or right arrow you get a movement of only one element (like the slida that has autoanimate on this page).

Autoanimate

For now when you use the autoanimate: true; (default is false) you get a self moving slider, one element by time and a hover-stop function which resumes on hover-out with the time left of the animation.

Using it

is pretty simple. Just embed JQuery latest and Slida.js on your page and follow the instructions bellow to set up you html, css and js code.

CSS

	#container {
	  max-width: 648px;
	  height: auto;
	  display: block;
	  overflow: hidden;
	}

	#handler{
	  width: 100%;
	  height: auto;

	.element{
	  display: block;
	  min-height: 325px; // what you want
	  width: 20px; // wont be the real widht you might wont to set this one as you wish :)
	  float: left;
	}

	// the transition class it must have the name transitions but you can change the settings as you wish
	.transitions {
	  -webkit-transition: all 1s ease;
	  -moz-transition: all 1s ease;
	  -o-transition: all 1s ease;
	  transition: all 1s ease;
	}

JS

Options

		autoanimate: false, // autoanimate?! (def false) if you add auto animate arrows dont work for now

		easing_type: 'swing', // easing type - choose from jquery easing or add another library ('swing')
        delay_time: 1000, // autoanimate delay

        width: 100,
		left_link: $('#link_right') // select by id left arrow
		right_link: $('#link_left') // select by id right arrow
